Installation & Setup

To use the Meme Generator library, you need to have Node.js installed. The library uses the canvas package for image manipulation.

Installation

npm install memecord

Basic Usage

const Meme = require('memecord');
const meme = new Meme();

// Generate a changemymind meme
async function generateMeme() {
    try {
        const buffer = await meme.changemymind('JavaScript is the best programming language');
        await fs.writeFile('output.jpg', buffer);
        console.log('Meme generated!');
    } catch (error) {
        console.error('Error:', error.message);
    }
}

generateMeme();
